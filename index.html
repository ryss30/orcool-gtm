Setup instructions
Claude
  1. Copy the MCP server URL above.
  2. Open Claude → Settings → Connectors → Add custom connector → paste the URL → Add.
  3. Sign in with the magic link (use the same Orcool account everywhere).
  4. Turn Orcool on in your chat (the tools menu), then run your prompt.
ChatGPT
  1. In ChatGPT, open the + (Plus) in the message box → More → Orcool.
  2. Sign in with the same Orcool email you use everywhere.
  3. Orcool tools are then available in your chats. A paid plan is steadier; on Free, prefer Claude or Cursor.
Cursor & other clients
  1. Copy the MCP server URL above.
  2. Add it as a custom MCP server in your client's MCP settings. In Cursor: Settings → MCP → Add new server → paste the URL.
  3. Sign in with the magic link (same Orcool account everywhere). Orcool tools are then available in your client.

Claude and Cursor are the smoothest first connect. ChatGPT works too, on a matched account (a paid plan is steadier). One rule that saves headaches: sign in with the same Orcool account in every client.

Questions

The honest answers.

Do I need to be an engineer?

No. Read the feeds in Slack and there's nothing to build. The MCP is for wiring the signal into your own systems. Plenty of producers start in Slack and move over later.

How does sign-in work?

Paste the MCP URL into your client. It opens a browser tab, you enter your email, and we send a one-click magic link. No password, no separate account to set up. After that the tools are live in every conversation.

Is the data real, or model guesswork?

Real. Live competitor creative across 40+ markets, structured and scored. The why on a pattern is grounded in what's running and holding in spend.

What does it cost?

Connecting is free, and your first report is free. All three feeds run about $250/mo, on a card, cancel anytime. The full loop with finished creative starts at $3K/mo plus a media share. We don't sell concepts on their own.

What can the MCP do beyond the three feeds?

Read-only covers competitors, patterns, trends, insights, and reports. Brand-write lets you onboard a brand, score concepts through its Lens, and run a full cycle.
